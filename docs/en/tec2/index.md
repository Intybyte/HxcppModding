I will mod the game [The Enchanted Cave 2](https://store.steampowered.com/app/368610/The_Enchanted_Cave_2/). 

The developer Dustinaux gave me some insights for modding the game:

- The game is written in Haxe
- It uses Lime lib
- Most stuff is defined in XML

The game is an RPG dungeon crawler with a basic crafting & enchanting system and some other features.

The objective is adding custom items to the game, each item has a picture, but for the sake of simplifying for now
we will target only adding items themselves and maybe recycle the image assets of other items.

This might create problems with the save data, so better take a backup.
